# Advanced HTML

Welcome!

This project focuses on learning how to structure a web page using HTML only. No CSS or styling is used, so the pages may look simple or "ugly" — this is expected.

## 🎯 Objectives

- Understand HTML5 structure
- Use semantic HTML tags
- Organize content properly
- Follow correct hierarchy with headings
- Create lists, tables, and embed media
- Structure a clean and valid HTML page

## ⚠️ Important

- Pay attention to details (lowercase, spelling, syntax)
- Code must be W3C compliant
- Use "Techium" as the company name in the project

## 📁 Project

All files are inside the `html_advanced` directory.
